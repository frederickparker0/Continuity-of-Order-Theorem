## Branch Boundaries

### Core Mathematics Candidate

This branch owns definitions: state, path, order, transition, boundary, comparator, functional, admissibility, and falsification. It does not borrow empirical success from any science branch until that branch passes its own gates.

### Turbulence

Current live public state: C338 repaired the target-side count and froze the correct `76+20 = 96` JHTDB `transition_bl` contract. The target side is local and hash-frozen. The source side still lacks admitted provenance. No scoring or theorem-grade promotion follows from target repair alone.

### Transition Front / Turbulence

The transition-front lane has a frozen single-domain turbulence protocol:

`TURBULENCE_SINGLE_DOMAIN_PROTOCOL_FROZEN_SPARSE_FRONT_NO_SCORE_NO_PROMOTION`.

The current lane is explicitly only:

`sparse point-path front surrogate + residual class-transition target + no-score claim boundary`.

It is not a true `Gamma_D` front with local normals, tangents, or residual normal motion. The protocol inherits the C337/C338 `76+20` contract, keeps the 20 target payloads as no-score target construction evidence, and remains blocked because the 76 source payload rows still need explicit-token reexport or byte-identical archived-bundle admission.

### Fracture / Crack Propagation

The fracture branch is real and substantial, but currently blocked. The C178 rerun found:

- official source identity and latest-version boundary can be discussed;
- the latest source is restricted;
- no identifiable access/grant/denial artifact is admitted;
- no complete local `H01.zip` matching the official MD5 target is admitted;
- schema extraction, replay, scoring, and theorem promotion are forbidden.

Next valid action: acquire an identifiable Zenodo `11669624` access artifact or the complete official-md5 `H01.zip` payload before any replay or scoring.

### Plasma Reconnection

The plasma branch has observed-data intake and a locked MMS reconnection scoring design, not a performance result.

The C93 gate requires event/control scoring with external event and control labels, event-level holdouts, at least 12 positive reconnection intervals and 12 controls, raw-channel baselines, label-shuffle controls, reversed-time controls, wrong-interval controls, and future-leak controls.

Until that event/control set exists, public claims stop at protocol lock.

### Protein Folding

Protein folding remains a diagnostic/reproduction branch. It has local Domainome evidence and a reproduction validator, but it needs independent reproduction and broader generalization before public promotion.

### Planck / Action-Phase

The Planck branch tests whether COT can recover Planck's constant as an invariant action-phase kernel:

`S = kappa_A phi`, `E = kappa_A omega`, `p = kappa_A k`, with `kappa_A = hbar`.

Current honest state:

- photoelectric calibration gives a close real-data estimate of `h`;
- Davisson-Germer gives an independent spatial/momentum check within about 1.23% using frozen `h`;
- Compton rounded primary data fails strict row-level prediction;
- source-grade audit blocks P9 promotion;
- no numerical derivation claim is allowed.

### DESI / Vacuum Energy / theta_V

This branch should be split mentally into DESI DR2 dark-energy protocol scaffolding and vacuum/theta_V reference-beat testing. The DESI branch is `protocol_created_pending_chain_import`. The theta_V lane has a frozen diagnostic definition and reference-beat contract, but no fill-ready candidate has beaten the reference.

### Failure and Gauntlet System

The failure branch is infrastructure. It tracks blockers, data walls, mutation failures, and claim-boundary repairs. It cannot promote COT by itself.
