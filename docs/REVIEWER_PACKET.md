# Reviewer Packet

This file is for reviewers who want the shortest honest route into the project.

## One-Sentence Summary

Continuity-of-Order is a theorem-candidate program testing whether ordered path histories preserve transition information that static or aggregate summaries can miss.

## What Is Being Claimed

- COT is a structured, falsifiable research program.
- Several branches have produced local diagnostics, validators, failure reports, and data-wall records.
- No public theorem-grade L6/L7 promotion is claimed.

## What Is Not Being Claimed

- COT does not currently solve turbulence.
- COT does not currently prove protein folding.
- COT does not currently prove vacuum energy.
- COT does not currently outrank established theory without remaining gates.

## Fastest Useful Review

1. Read `THEOREM_CANDIDATE.md`.
2. Read `MATHEMATICS_CANDIDATE.md`.
3. Read `EVIDENCE_STATUS.md`.
4. Pick one branch and verify whether the blocker is real.
5. Open an issue using the relevant template.

## How to Challenge the Work

The best challenges are specific:

- show evidence leakage,
- show a hidden coefficient fit,
- show a missing baseline,
- show a mutation control that should fail but does not,
- show a dataset provenance problem,
- show that a branch is only baseline-equivalent,
- show that the formal definitions are insufficient.

## What Would Be a Major Contribution

- A clean source-provenance repair for the turbulence branch.
- An independent protein-folding reproduction.
- A strict vacuum/theta_V candidate test that beats the frozen reference.
- A formal theorem candidate with exact objects and falsification criteria.
- A negative result that prevents overclaiming.
