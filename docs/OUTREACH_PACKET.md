# Outreach Packet

Research originator and project steward: **Frederick M. Parker**.

This file gives short, honest descriptions for sharing the project without overstating it.

## One Sentence

Continuity-of-Order is a theorem-candidate research program testing whether ordered path histories reveal transition signals that static snapshots or aggregate summaries miss.

## Short Public Description

Continuity-of-Order (COT) is a public theorem-candidate framework by Frederick M. Parker. It tests whether ordered histories can expose transitions, failures, or boundary changes across turbulence, transition-front, fracture, plasma, protein-folding, Planck/action-phase, and DESI/vacuum-energy branches. The repository includes claim boundaries, failure logs, data walls, reproducibility routes, and contribution templates.

## Longer Public Description

COT does not claim to solve turbulence, protein folding, plasma reconnection, fracture propagation, Planck's constant, or vacuum energy. Instead, it asks a narrower testable question: can a frozen path-history functional recover or constrain transitions better than static, pointwise, or aggregate baselines under observed-data rules?

The public repository is designed for criticism and continuation. Each branch lists its current evidence, blocker, next gate, and forbidden claims. Failed gates are preserved as first-class evidence.

- COT has several branch protocols and evidence gates.
- COT is being published for open testing, critique, reproduction, and falsification.
- The strongest value of the repository is its strict claim boundary and failure ledger.
