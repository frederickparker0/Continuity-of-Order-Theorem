# Thesis Statement

Research originator and project steward: **Frederick M. Parker**.

## Provisional Thesis

Continuity-of-Order (COT) is a theorem-candidate framework for testing whether ordered histories preserve transition information that static, pointwise, or aggregate summaries can erase.

At present, this thesis is **not proven**. It is supported only as a disciplined research program with several local diagnostic signals, strict failure boundaries, and exact data-wall requirements.

## Positive Claim

COT has value as a validation discipline when it:

1. forces path/history definitions to be explicit;
2. separates target construction from scoring;
3. prevents same-source and same-anchor leakage from becoming theorem claims;
4. treats baseline absorption and mutation survival as decisive;
5. records failures as structured evidence;
6. demands observed/source-admitted data before promotion.

## Negative Claim

COT should not currently be advertised as:

- a turbulence solution,
- a protein-folding solution,
- a vacuum-energy proof,
- a promoted theorem.

## Current Research Question

The live question is:

> Can a frozen COT path/history object recover or predict heldout transition structure while surviving source provenance, baselines, wrong-form mutations, uncertainty gates, and independent reproduction?

The current answer is:

> Not yet. The next honest work is to close the exact data/provenance gates and then run locked scoring attempts.
