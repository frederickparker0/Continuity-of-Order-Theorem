# Continuity-of-Order Research Compendium

Research originator and project steward: **Frederick M. Parker**.

This repository is a public-facing compendium of the COT research work across the active branch family:

- core mathematics candidate,
- turbulence transition-path diagnostics,
- transition-front turbulence protocol,
- fracture / crack-propagation gates,
- plasma reconnection gates,
- protein-folding diagnostic/reproduction gates,
- Planck / action-phase recovery gates,
- DESI / vacuum-energy / theta_V diagnostic constant gates,
- failure and gauntlet infrastructure.

The central working idea is **Continuity of Order (COT)**: ordered path history may contain transition information that pointwise, aggregate, or static summaries can miss.

## Honest Status

This is **not** a solved-theorem claim.

No public L6/L7 theorem promotion is claimed in this package. The current work is best described as a theorem-candidate program with strict evidence gates, observed-data discipline, failure logs, and data-wall requests.

The strongest current state is:

- the turbulence branch has repaired a major target-payload data wall, but remains blocked on source provenance and scoring firewalls;
- the transition-front turbulence lane has a frozen sparse-front no-score protocol, but not true geometric front-motion evidence;
- the fracture branch has a grounded proof-obligation rerun, but is blocked at access/payload admission;
- the plasma branch has an MMS reconnection intake/protocol lock, but no event-control score;
- the protein-folding branch has local diagnostic signals and an external-reproduction validator, but public promotion is blocked;
- the Planck/action-phase branch has partial recovery evidence, but no derivation or promotion claim;
- the DESI/vacuum-energy branch has strict protocols/contracts, but no candidate has yet beaten the frozen reference baseline.

## Public Claim Boundary

Allowed:

- COT is a structured research program for testing whether ordered histories carry transition signals.
- The work has produced reproducible contracts, validators, failure ledgers, data-wall specifications, and protocol locks.
- Several branches have diagnostic, local recovery, or protocol evidence.

Forbidden:

- COT solves turbulence.
- COT proves transition-front prediction.
- COT validates fracture propagation.
- COT validates plasma reconnection.
- COT proves protein folding.
- COT derives Planck's constant from first principles.
- COT proves vacuum energy.
- Any branch has earned theorem-grade promotion without the remaining locked gates.

## Data Policy

Raw datasets, large archives, local machine paths, private credentials, and working automation clutter are not included here. Public reproduction should use the listed source datasets and the frozen contracts described in the compendium.

## How to Continue the Work

The fastest way to help is to open one focused issue:

- data wall or source request,
- replication attempt,
- mathematics formalization,
- failure report,
- claim correction.

Useful contributions make the project harder to fool. A failed replication, a sharper definition, or a corrected overclaim can be as valuable as a pass.
