# Continuity-of-Order Research Compendium

This repository is a public-facing compendium of the COT research work across three active branches:

- turbulence transition-path diagnostics,
- protein-folding diagnostic/reproduction gates,
- vacuum-energy / theta_V diagnostic constant gates.

The central working idea is **Continuity of Order (COT)**: ordered path history may contain transition information that pointwise, aggregate, or static summaries can miss.

## Honest Status

This is **not** a solved-theorem claim.

No public L6/L7 theorem promotion is claimed in this package. The current work is best described as a theorem-candidate program with strict evidence gates, observed-data discipline, failure logs, and data-wall requests.

The strongest current state is:

- the turbulence branch has repaired a major target-payload data wall, but remains blocked on source provenance and scoring firewalls;
- the protein-folding branch has local diagnostic signals and an external-reproduction validator, but public promotion is blocked;
- the vacuum-energy branch has a strict reference-beat contract, but no candidate has yet beaten the frozen reference baseline.

## Public Claim Boundary

Allowed:

- COT is a structured research program for testing whether ordered histories carry transition signals.
- The work has produced reproducible contracts, validators, failure ledgers, and data-wall specifications.
- Several branches have diagnostic or local recovery evidence.

Forbidden:

- COT solves turbulence.
- COT proves protein folding.
- COT proves vacuum energy.
- Any branch has earned theorem-grade promotion without the remaining locked gates.

## Data Policy

Raw datasets, large archives, local machine paths, private credentials, and working automation clutter are not included here. Public reproduction should use the listed source datasets and the frozen contracts described in the compendium.
