# Promotion Ladder

The COT work uses promotion language to prevent overclaiming.

## Levels

| Level | Meaning | Public Claim Allowed |
|---|---|---|
| L0-L2 | Definitions, data locks, protocols, local scaffolds | Preparatory work only |
| L3-L4 | Local diagnostic or construction evidence | Diagnostic result |
| L5 | Multi-benchmark recovery / known-result reproduction | Recovery framework |
| L6 | Clean heldout recovery with no hidden reuse or retuning | Heldout recovery |
| L6+ | Heldout recovery plus diagnostic utility | Heldout diagnostic framework |
| L7 | Distinct prediction/mechanism survives baselines and mutations | Distinct evidence |
| L8+ | Broad theorem-grade support across independent settings | Theorem-level candidate |

## Current Promotions

No branch currently claims public L6/L7 promotion in this package.

## Promotion Kill Rules

A promotion attempt is killed by:

- hidden coefficient or residual repair after failure,
- same evidence used for construction and scoring,
- missing source provenance,
- missing uncertainty or tolerance,
- baseline tie/loss,
- wrong-form mutation survival,
- point-path evidence promoted to region-support evidence,
- vague data substitution,
- claim-language drift.
