# Continuity-of-Order Theorem Candidate

Research originator and project steward: **Frederick M. Parker**.

## Informal Statement

Systems that appear chaotic, turbulent, unstable, or failure-driven may still preserve order in their histories. If that order is tracked as a path, rather than as isolated states or aggregate summaries, it may reveal transition structure before the transition is obvious in ordinary measurements.

This is the working Continuity-of-Order theorem candidate.

## Core Conjecture

Let a physical or empirical system be observed as an ordered path of states:

`P = (x_0, x_1, ..., x_n)`

where each state is source-admitted, observed, or reproducibly derived from observed/source-admitted data.

If a transition or failure boundary exists in the system, then there may exist a path-history functional:

`COT(P)`

that detects, ranks, or constrains the transition better than a pointwise or aggregate summary:

`S(x_i)` or `A(P)`

provided that the COT functional survives:

1. heldout source separation,
2. source-byte provenance,
3. strong physical/statistical baselines,
4. wrong-form and mutation controls,
5. uncertainty/tolerance gates,
6. no hidden coefficient or residual repair,
7. scope discipline between point-path and region-level claims.

## Current Status

This is not yet a theorem.

It is currently a theorem-candidate program with active evidence gates. The work has produced diagnostic packets, protocol locks, local signals, failure logs, data-wall requests, and reproducibility contracts, but no public L6/L7 theorem promotion is claimed.

For the formal route from this working idea to a mathematics candidate, see [Route to a Mathematics Candidate](MATHEMATICS_CANDIDATE.md). For the branch map, see [COT Branch Index](BRANCH_INDEX.md).

## Branch Interpretations

### Turbulence

COT asks whether source-followed path history in turbulence contains transition information missed by static or aggregate velocity/pressure summaries.

Current state: future-material target payloads are repaired, but source provenance and scoring firewalls remain blocked.

### Transition Front / Turbulence

COT asks whether a transition front can be read as ordered path history without falsely promoting sparse point anchors into a true geometric front.

Current state: a single-domain turbulence transition-front protocol is frozen as a sparse-front residual class-transition adapter. It is no-score and no-promotion until source admission is repaired.

### Fracture / Crack Propagation

COT asks whether ordered crack-growth paths can detect or constrain propagation boundaries under source-admitted fracture data.

Current state: proof-obligation rerun remains blocked at access/payload admission; no scoring or promotion.

### Plasma Reconnection

COT asks whether causal path features in MMS plasma data can recover reconnection events beyond raw magnetic, density, and flow baselines.

Current state: intake and scoring protocol are locked; event/control scoring set is not yet admitted.

### Protein Folding

COT asks whether ordered folding/stability evidence contains diagnostic structure missed by static protein summary features.

Current state: local diagnostic signals and an external validator exist, but public promotion is blocked by independent reproduction and generalization.

### Planck / Action-Phase

COT asks whether action-phase order can recover Planck's constant as `kappa_A = hbar` under real-data recovery tests.

Current state: partial recovery evidence exists, but strict prediction/source-grade promotion is blocked. No derivation claim is allowed.

### DESI / Vacuum Energy / theta_V

COT asks whether a frozen diagnostic constant can beat a reference baseline under strict heldout constraints.

Current state: DESI protocol scaffolding exists and the theta_V reference-beat contract exists, but no candidate has yet beaten the frozen reference.

## Claim Boundary

Allowed claim:

> COT is a path-history theorem candidate with strict observed-data gates and several active diagnostic branches.

Forbidden claims:

> COT has solved turbulence, proved transition-front prediction, validated fracture propagation, validated plasma reconnection, proved protein folding, derived Planck's constant, or proved vacuum energy.

The purpose of this repository is to make the theorem candidate public enough to be criticized, reproduced, and improved without overstating the evidence.
